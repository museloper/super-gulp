# Super Gulp

Learn Gulp!

```bash
$ npm i gulp-cli -g
$ git init
# git remote add origin [URL]
$ npm init
$ npm i gulp -D
# package.json 수정
$ npm i @babel/register @babel/core @babel/preset-env -D

# Plugins...
$ npm i gulp-pug -D
$ npm i del -D
$ npm i gulp-image -D # 예제에서 6.2.1 버전 사용해야 오류가 발생하지 않음
$ npm i node-sass gulp-sass -D
$ npm i gulp-autoprefixer -D
$ npm i gulp-csso -D
$ npm i gulp-bro -D
$ npm i babelify -D
$ npm i uglifyify -D

# webserver
$ npm i gulp-webserver -D

# deploying
$ npm i gulp-gh-pages -D
```
